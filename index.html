🌱 Feed the seed (local intelligence)
Seed = local Q&A + rules which AURA-X Ω uses before calling any LLM.

Here the idea is simple: you plant small, clear Q&A “seeds” about AURA-X Ω, your projects, values and stories. Whenever a new TM input comes in, the engine first tries to answer from these seeds. Only if nothing matches, it asks the offline LLM.

In this prototype, seeds are hard-coded inside JavaScript (an array of Q&A). In the next step you can connect this panel to a file or database.
